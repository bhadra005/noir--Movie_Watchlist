Abstract

In this project, we explore the user interface of a Personal Movie Tracker Website which aims at improving user engagement with movies through an easy discovery and preference management system. The user's site includes the key pages as follows:

Home Page: Contains the Site name and logo, background image, featured movies, and navigation buttons.
Settings/Profile Page: Notification settings, profile updates, and genre preferences.
Browse Movies Page: Displays movies in a grid format and has various filters like search bar and hover effects.
Watchlist Page: Enables users to manage saved films, set priority tags, and sort entries.
Movie Details Page: Displays movie information and an option to add the film to the watchlist.
About/Team Page: Displays the goals of the project, team bios, the tech stack, screenshots, and credits.

The site design was responsive, straightforward, and aesthetically pleasing for optimal user engagement with the movie tracking experience.
